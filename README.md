# Log linter

- Выполнены требования к линтеру и технические требования. 
- Реализованы авто-исправления для lowercase и nospecials
- Приложение поддерживает настройку чувствительных данных через конфиг .golangci.yaml
- Добавлен ci для автоматического тестирования

### Сборка приложения
`make build`

### Запуск линтера
`make lint`
